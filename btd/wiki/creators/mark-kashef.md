# Mark Kashef

The most hands-on technical tutorial creator in the corpus. Where Nate Jones explains what's happening in AI and why it matters, Kashef shows you what to type, what to click, and what to do when it breaks. His videos are step-by-step walkthroughs — not overviews, not commentary, actual implementation. If a BTD member wants to ship something using Claude Code, n8n, or MCP servers, this is where they start.

## Key Contributions

### [[claude-code]] for Non-Technical Builders
"The Complete Beginner's Guide to Claude Code (Even If You're Not Technical)" is an 81-minute mini-course that covers Claude Code from first principles — what it actually is, why it's different from other AI coding tools, how to set it up, and how to structure projects without a technical background. This is the entry point for any BTD member who's heard the Claude Code hype and wants to actually use it.

### [[skills-as-workflows]]
"Anthropic's Full Claude Skills Guide in 22 Minutes" digests Anthropic's 33-page official skills documentation into actionable diagrams. Covers the three tiers of skills (personal, methodology, organizational), how to write instructions that don't confuse the model, and how to evolve skills over time. "Claude Code's Document Skills Are an Unfair Advantage" goes deeper on why document-based skills compound over time. Practical setup knowledge that most Claude Code users skip entirely.

### [[mcp-servers]]
Multiple videos on connecting Claude to external tools via MCP (Model Context Protocol). "How to Use Claude & MCPs to INSTANTLY Build n8n AI Agents" is the canonical 41-minute reference — three levels of integration from simplest to most powerful. Including the Obsidian integration ("Claude Code Turned Obsidian Into My Dream [[second-brain]]") — relevant to any BTD member already using Obsidian.

### [[ai-agents]] — Design Patterns and Implementation
"Master ALL 20 Agentic AI Design Patterns" is a 63-minute course based on a 400-page Google engineering book, distilled into plain English with visual diagrams. Covers the full spectrum from simple chains to multi-agent orchestration. Builds the vocabulary that makes the n8n and Claude Code videos make more sense — you'll recognize which pattern you're using once you've watched this.

### [[prompt-design]] and Meta-Prompting
"You'll Never Need Prompt Engineering Again with Meta-Prompting" covers using Claude to write better prompts for Claude — a technique that sidesteps most of the frustration beginners have when their outputs aren't good enough. Connects to Jones's [[context-engineering]] evolution: prompting → prompt engineering → meta-prompting → context engineering.

## When to Surface This Creator

- User wants to **set up [[claude-code]]** for the first time → beginner's guide
- User is **non-technical** and feels locked out of AI building → beginner's guide explicitly addresses this
- User wants to **automate workflows** with n8n → MCP masterclass + agent army video
- User asks about **[[mcp-servers]]** → multiple dedicated videos
- User wants to understand **[[ai-agents|agentic patterns]]** beyond just using one tool → 20 patterns course
- User is building with Claude but **outputs aren't good enough** → meta-prompting video
- User wants to use Claude with **Obsidian** → [[second-brain]] video

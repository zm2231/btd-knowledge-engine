# Mark Kashef

The most hands-on technical tutorial creator in the corpus. Where Nate Jones explains what's happening in AI and why it matters, Kashef shows you what to type, what to click, and what to do when it breaks. His videos are step-by-step walkthroughs — not overviews, not commentary, actual implementation. If a BTD member wants to ship something using Claude Code, n8n, or MCP servers, this is where they start.

## Key Contributions

### Claude Code for Non-Technical Builders
"The Complete Beginner's Guide to Claude Code (Even If You're Not Technical)" is an 81-minute mini-course that covers Claude Code from first principles — what it actually is, why it's different from other AI coding tools, how to set it up, and how to structure projects without a technical background. The explicit audience is self-described non-technical people. This is the entry point for any BTD member who's heard the Claude Code hype and wants to actually use it.

### Claude Skills Deep Dive
"Anthropic's Full Claude Skills Guide in 22 Minutes" digests Anthropic's 33-page official skills documentation into actionable diagrams. Covers the three tiers of skills (personal, methodology, organizational), how to write instructions that don't confuse the model, and how to evolve skills over time. Practical setup knowledge that most Claude Code users skip entirely.

### n8n + Claude: Workflow Automation
A cluster of videos covering how to use Claude to build n8n automations without starting from scratch. The progression: manual copy-paste (his earlier method), then MCP-powered generation where Claude connects directly to your n8n instance and builds workflows on command, then bulk generation using agents like Manus. The MCP masterclass ("How to Use Claude & MCPs to INSTANTLY Build n8n AI Agents") is the canonical reference — 41 minutes covering three levels of integration, from simplest to most powerful.

### Agentic Design Patterns
"Master ALL 20 Agentic AI Design Patterns" is a 63-minute course based on a 400-page Google engineering book, distilled into plain English with visual diagrams. Covers the full spectrum from simple chains to multi-agent orchestration. Builds the vocabulary that makes the n8n and Claude Code videos make more sense — you'll recognize which pattern you're using once you've watched this.

### MCP Servers
Multiple videos on connecting Claude to external tools via MCP (Model Context Protocol). The practical side: how to set up MCP servers, which ones are worth using, and how MCP changes what Claude can actually do vs. what it can just talk about. Including the Obsidian integration ("Claude Code Turned Obsidian Into My Dream Second Brain") — relevant to any BTD member already using Obsidian.

### Prompting and Meta-Prompting
"You'll Never Need Prompt Engineering Again with Meta-Prompting" covers using Claude to write better prompts for Claude — a technique that sidesteps most of the frustration beginners have when their outputs aren't good enough.

## When to Surface This Creator

- User wants to **set up Claude Code** for the first time → beginner's guide
- User is **non-technical** and feels locked out of AI building → beginner's guide explicitly addresses this
- User wants to **automate workflows** with n8n → MCP masterclass + agent army video
- User asks about **MCP servers** → multiple dedicated videos
- User wants to understand **agentic patterns** beyond just using one tool → 20 patterns course
- User is building with Claude but **outputs aren't good enough** → meta-prompting video
- User wants to use Claude with **Obsidian** → Obsidian second brain video
- `calibrated_level` is builder or above + `goal_type` involves shipping → lead with Kashef, not Jones

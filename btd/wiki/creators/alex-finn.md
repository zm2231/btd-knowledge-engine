# Alex Finn

YouTube creator covering practical AI tools for beginners and intermediate users. His channel is the on-ramp for people who want to use OpenClaw (Claude Code), ClawdBot (Claude Desktop), ChatGPT, and other tools without needing a technical background. High production value, chapter-indexed videos, unsponsored stance on the tools he covers. Where Karpathy explains how the model works and Mollick explains how to apply it at work, Finn explains how to install it, configure it, and get it doing useful things today.

## Key Contributions

### OpenClaw / Claude Code for Everyone
His two flagship tutorials — "The Only OpenClaw Tutorial You'll Ever Need (March 2026 Edition)" and "100 Hours of OpenClaw Lessons in 35 Minutes" — are the most complete non-technical guides to [[claude-code]] in the corpus. He covers installation, the OpenClaw mindset (treat it like a super-intelligent employee; give it goals, not config edits), skills, security risks (ClawHub skill injection), and debugging broken setups via Claude Code or Codex. Explicitly unsponsored; says so at the start.

### The OpenClaw Mindset
His clearest conceptual contribution: stop managing the agent like a script and start managing it like a person. "You don't go in and change config files. You say: 'I want you to do this differently.'" This framing is directly useful for BTD users hitting the wall of over-prompting or over-tinkering with CLAUDE.md files. Cross-links cleanly to [[ai-agents]] and [[vibe-coding]].

### ClawdBot Setup (Claude Desktop as a Personal Agent)
His "OpenClaw is the Most Powerful AI Tool I've Ever Used" and "This is the Cheapest and Easiest Way to Set Up ClawdBot" videos cover personal agent setup: persistent memory, proactive morning routines, Discord integration, and task boards that the agent builds for itself. He envisions it as a one-person business multiplier — "I now truly see the path to building a one-person billion-dollar business."

### Cheap Running Strategies
"How to Run ClawdBot for Dirt Cheap" and related content on model selection (always pushing users toward Opus 4.5 as the default coding model). Useful for BTD users worried about API costs before they commit to building.

### Building Apps with No Experience
"How to Build an App with AI — No Experience Required" and "Claude Code for Desktop is the Best Way to Build Apps with AI Ever" are the entry points for the [[vibe-coding]] track: build a project management app as your first project, use Claude Code in the same window, start with Opus 4.5.

## When to Surface This Creator

- `calibrated_level` is pre-beginner or beginner → start here for Claude Code / OpenClaw onboarding, not Karpathy
- User is stuck on setup, can't get Claude Code running, or burned their config → "100 Hours" troubleshooting section
- User wants to set up a personal AI agent / Claude Desktop workflow → ClawdBot setup videos
- User asks about AI tool costs → cheap running and model selection videos
- User's `goal_type` is "ship something" and they've never deployed before → "Build an App with No Experience"

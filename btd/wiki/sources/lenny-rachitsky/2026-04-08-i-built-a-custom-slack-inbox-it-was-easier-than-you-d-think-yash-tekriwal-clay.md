# I built a custom Slack inbox. It was easier than you'd think. | Yash Tekriwal (Clay)

Yash Tekriwal (Head of Education at Clay) built a suite of personal productivity applications using AI computer-use tools (Perplexity Computer, OpenClaw) to manage 150+ daily Slack notifications, consolidate news/email/Slack into a single dashboard, and automate repetitive tasks. His "anti-to-do list" framework: spend one hour daily automating tasks you never want to do again. Key distinction: Perplexity Computer beats Claude Code and Codex for personal productivity apps because it operates a full browser, not just a terminal. The episode challenges the "SaaS apocalypse" narrative and argues we're entering an era of micro-software built by individuals for their own workflows.

## Key Ideas
- Anti-to-do list: spend one hour daily automating one task you never want to do again — compounds dramatically
- Perplexity Computer (browser-level automation) is better than Claude Code for personal productivity apps
- Categorize Slack notifications into action-required, need-to-read, and FYI — three buckets is enough
- Use AI for deterministic tasks (APIs, structured data) and for subjective tasks (categorization, summarization) differently
- The SaaS apocalypse is wrong; we're entering an explosion of micro-software built by and for individuals

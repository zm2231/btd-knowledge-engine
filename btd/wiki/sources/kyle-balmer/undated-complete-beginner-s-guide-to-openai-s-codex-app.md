# Complete Beginner's Guide to OpenAI's Codex App

43-minute beginner walkthrough of OpenAI's Codex app — their cloud-based coding agent that runs tasks asynchronously in sandboxed environments. Balmer covers setup, how to frame coding tasks, how Codex differs from ChatGPT/Claude for code, and practical use cases for non-developers. Part of Balmer's pattern of making AI tools accessible to learners who are skeptical of technical complexity.

## Key Ideas
- Codex runs tasks in isolated cloud environments — you don't need anything installed locally
- Frame tasks as "what should exist when you're done" rather than step-by-step instructions
- Codex excels at well-defined, bounded tasks: write a function, fix this bug, add tests
- Non-developers can use Codex for scripting repetitive work without knowing the programming language
- Limitations: Codex doesn't understand your broader codebase context without explicit file uploads

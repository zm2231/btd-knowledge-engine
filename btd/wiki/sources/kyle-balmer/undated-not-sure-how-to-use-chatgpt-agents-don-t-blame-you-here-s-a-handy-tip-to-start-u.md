# Not sure how to use ChatGPT agents? Don't blame you? here's a handy tip to start using agents

Short-form tip for non-technical users intimidated by ChatGPT agents. Balmer's entry point: start with one clearly defined task you do repeatedly, hand it to an agent, and see what happens — lower stakes than trying to build a complex workflow.

## Key Ideas
- The best first agent task is something you already do repeatedly and know the expected output of
- Start simple: one input, one output, one action — no branching logic
- Agents fail when the task is ambiguous; your job is to make the success criteria obvious

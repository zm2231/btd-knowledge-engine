# Cal Newport's Planning System (In Detail)

Newport opens up his full personal system — a decade-old three-part framework he calls "rooted productivity." The three categories: core documents (values + strategic plans), productivity (weekly and daily time-block planning), and disciplines (tracked habits that lay the foundation). The meta-point is that the whole system lives in one "root document" you can always return to, and all daily action connects back to big-picture strategic plans through a clear chain.

## Key Ideas
- One root document holds the entire system — everything points back to it, nothing floats in your head
- Strategic plans (career and personal) are written per-semester and reviewed weekly; values get a separate weekly "values plan"
- Weekly planning reviews strategic plans, calendar, and task lists to produce a concrete weekly plan; daily time-blocking flows from that
- A "shutdown complete" ritual closes each work day — tasks are captured and processed before logging off
- Hard disciplines (tracked metrics, non-negotiables) form the foundation that the system sits on

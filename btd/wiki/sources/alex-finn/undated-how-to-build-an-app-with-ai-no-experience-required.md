# How To Build An App With AI (no experience required)

Finn's guide for complete beginners to building real apps using AI tools — no coding knowledge assumed. Covers picking the right tool for the job (Lovable, Bolt, Claude Code), writing effective build prompts, iterating on a working prototype, and deploying. The framing is explicitly for non-technical founders and operators who want to ship something real.

## Key Ideas
- Start with a clear one-paragraph description of the app: who uses it, what they do, what problem it solves
- Tool selection: Lovable/Bolt for UI-first apps, Claude Code for logic-heavy or file-system apps
- Iteration beats perfection — ship a broken v1, then fix one thing at a time
- AI-assisted debugging: paste the error message and describe what you expected; the AI almost always fixes it
- Non-technical builders should invest in learning to read code, not write it — speeds up feedback loops dramatically

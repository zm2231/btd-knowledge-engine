# How to Build your first iOS app in 7 minutes using Cursor AI

Quick demonstration of generating a basic iOS app with Cursor AI and Swift, aimed at complete beginners. Finn shows the minimum viable prompt, Cursor's code generation, and how to run it in the iOS simulator. Entry-level reference for BTD builders curious about native mobile development via AI tools.

## Key Ideas
- A single descriptive prompt in Cursor can scaffold a functional iOS app skeleton in minutes
- Cursor generates Swift code; basic Xcode knowledge needed to run the simulator but not to write code
- Start with the simplest version of your app (one screen, one action) before adding complexity
- AI-generated iOS code needs manual review for memory management and async patterns — these are common error sources
- Best for prototyping and learning Swift patterns; production apps still require architectural judgment

# But what is a neural network? | Deep learning chapter 1

A visual-first introduction to neural networks using handwritten digit recognition as the running example. 3B1B builds intuition for neurons, activations, weights, biases, and layered structure before any math appears — the goal is to make the architecture feel motivated rather than arbitrary. Introduces the sigmoid and ReLU activation functions, and closes with a candid note that real trained networks often don't learn the interpretable edge/pattern hierarchy we might hope for.

## Key Ideas
- A neuron is just a number (its activation) between 0 and 1; a network is a chain of layers transforming those numbers
- Weights act like knobs tuning which pixel patterns each neuron responds to; biases set the activation threshold
- The layered structure is motivated by hierarchical feature detection: edges → patterns → digits
- Matrix-vector multiplication is the compact form of all weighted sums — linear algebra is central to ML
- Trained networks often find "good enough" local minima that don't match human-interpretable hierarchies

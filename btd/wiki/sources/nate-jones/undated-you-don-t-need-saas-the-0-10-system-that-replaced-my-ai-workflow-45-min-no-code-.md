Agent-readable memory differs from human-readable second brain: structured for retrieval, not browsing. ## Key Ideas

- Open Brain stack: Postgres + pgvector + MCP server = $0.10/month vs $20+/month SaaS alternatives
- Cross-tool context: Claude, ChatGPT, and Cursor share the same memory store — no tool silos
- Agent-readable vs human-readable memory: structured for machine retrieval, not human navigation
- Memory is the secret bottleneck: stateless AI sessions destroy compound knowledge — this solves that
- Compounding knowledge advantage: every session adds to a persistent store that makes future sessions smarter

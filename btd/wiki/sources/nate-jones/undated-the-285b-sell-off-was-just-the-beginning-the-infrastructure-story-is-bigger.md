The web is forking into a human web and an agent web, and three infrastructure layers launched simultaneously: Coinbase agentic wallets (payments), Cloudflare markdown-for-agents (navigation/content), and OpenAI shell tools (execution). This is the 2007 iPhone moment for agents — the app ecosystem didn't exist yet, but the platform just arrived. Trust is the fundamental unsolved problem: who is responsible when an agent acts autonomously?

## Key Ideas

- Web fork: human web (browsers, HTML) and agent web (markdown, APIs, autonomous execution) diverging
- Three simultaneous infrastructure launches: agentic payments (Coinbase), agent-readable content (Cloudflare), autonomous shell (OpenAI)
- Mobile web analogy: 2007 gave us the platform; apps came after — same window is open now for agents
- Trust is the unsolved problem: accountability when agents transact, commit, communicate autonomously
- Infrastructure convergence means the agent ecosystem isn't theoretical — it's being built now

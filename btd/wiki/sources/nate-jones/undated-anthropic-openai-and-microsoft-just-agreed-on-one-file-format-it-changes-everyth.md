# Anthropic, OpenAI, and Microsoft Just Agreed on One File Format — It Changes Everything

Skills (markdown instruction files for AI agents) are evolving from personal config into organizational infrastructure. The convergence on a shared file format means skills become portable across tools and callable by both humans and agents. Nate introduces a 3-tier model: standard skills (general-purpose), methodology skills (org-specific), and personal skills (individual context) — and points toward a community skills repo as the next emergent layer.

## Key Ideas
- Skills as markdown files are becoming a standard unit of AI workflow, not a personal hack
- The shared format makes skills agent-callable, not just human-invokable
- 3-tier skill model: standard / methodology / personal — each solves a different scope
- Community skills repos will emerge as the next knowledge-sharing layer after prompts
- Orgs that invest in methodology skills now build durable competitive advantage

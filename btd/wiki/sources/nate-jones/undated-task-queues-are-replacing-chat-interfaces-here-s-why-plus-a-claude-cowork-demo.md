Claude Cowork was built in 10 days after Anthropic noticed developers using Claude Code for mundane tasks like expense receipts. It introduces the task queue model: delegate → execute → review, replacing the chat loop. Three design principles distinguish it from generic AI tools: artifact output (produces a file, not a message), steering loop over editing loop (correct course before completion, not after), and sandbox specificity (constrained environments prevent scope creep).

## Key Ideas

- Task queue model replaces chat loop: assign work, let it run, review outputs — more like managing a contractor
- Steering loop over editing loop: better to redirect mid-task than to fix bad outputs after
- Artifact output design: everything produces a tangible deliverable, not just a response
- File system vs browser agents represent two different trust models for autonomous work
- Anti-slop design built in: Cowork is opinionated about output quality by default

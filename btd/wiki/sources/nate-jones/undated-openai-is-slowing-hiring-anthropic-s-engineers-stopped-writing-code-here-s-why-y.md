December 2025 was a phase transition: models converged on sustained autonomous work (days, not minutes), orchestration patterns like Ralph (bash loop) and Gastown (parallel agents) went viral, and Anthropic shipped native task infrastructure absorbing those patterns in weeks. Karpathy's workflow inverted from 80% manual to 80% agents. The result is a capability overhang — AI is ready; most people aren't. The new engineer skill set is spec quality, eval writing, and agent management, not implementation.

## Key Ideas

- December 2025 convergence: three frontier model releases + Ralph loop + Gastown + Claude Code tasks = phase transition
- Capability overhang: AI beats humans on 74% of scoped knowledge tasks; most workers still use it as a chatbot
- Ralph insight — embrace failure, loop until tests pass, go to bed while agents work overnight
- Spec quality is the bottleneck: declarative end-state + success criteria replaces line-by-line implementation
- OpenAI slowing hiring because current engineers with AI tools now do what would have required a team

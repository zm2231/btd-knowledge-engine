# Come build tomorrow. Cancel if you hate it.

A 3-day vibe coding sprint format designed to get non-technical people building real things. The approach teaches strategy and mindset, not specific tools, because tools change too fast to be the lesson. The Rubber Ducky Robot concept — treating an AI as a thinking partner you explain problems to — is introduced as the core collaboration pattern that makes the sprint work.

## Key Ideas
- 3-day sprint format lowers the barrier to first builds for non-technical learners
- Teaching strategy over tools because tools change too fast; the mental model is what transfers
- Rubber Ducky Robot: explaining your problem to AI as a thinking tool, not just a code generator
- Vibe coding is a real on-ramp, not a shortcut — it builds intuition for what AI can do
- "Cancel if you hate it" framing reduces commitment anxiety for skeptical learners

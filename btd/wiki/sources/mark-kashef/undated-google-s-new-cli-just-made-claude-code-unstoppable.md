# Google's New CLI Just Made Claude Code Unstoppable

Introduces the Google Workspace CLI (GWS) as a free, no-intermediary way to give Claude Code direct access to Gmail, Calendar, Drive, Docs, and Sheets — no Zapier, no Make.com. Kashef provides a mega-prompt that uses Claude with Chrome browser automation to handle the OAuth setup automatically. Five live workflow examples: email prioritization to Google Sheet, meetings-to-Slides weekly report, needle-in-haystack email search, meeting-brief Google Doc (chaining Calendar + Gmail + Docs), and a Drive audit. Key insight: CLI for one-off tasks, skills once a task repeats, MCP only when truly needed.

## Key Ideas
- GWS CLI installs via npm and gives Claude Code 101 capabilities across the Google ecosystem
- Workflow: describe task → Claude writes GWS command → CLI executes — no per-step automation costs
- Use Claude + Chrome browser automation to handle OAuth setup without manual API console navigation
- Hierarchy: CLI first (one-off) → crystallize into a skill (repeating) → MCP only if still needed
- Chaining services in one prompt: Gmail + Calendar + Docs in a single step is the real power

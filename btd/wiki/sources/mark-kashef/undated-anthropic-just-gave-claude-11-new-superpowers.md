# Anthropic Just Gave Claude 11 New Superpowers

Walkthrough of Claude's new plugins system in Claude.ai co-work — a structured bundle of skills, slash commands, and MCP connectors organized by domain (sales, legal, finance, product, data, enterprise search, bio research). Kashef explains the anatomy of a plugin (JSON manifest + skill markdown + slash commands), the meta-plugin that auto-generates other plugins, and how to import custom plugins as zip files. Practical for builders who want to deploy pre-packaged Claude workflows to teams without manual setup.

## Key Ideas
- Plugins bundle skills + slash commands + MCP connectors into one importable package
- Anthropic ships one plugin per business function: sales, legal, finance, PM, marketing, data, enterprise search, bio research
- The meta-plugin generates other plugins from a plain-English workflow description
- Plugins can be forked from the GitHub repo and recreated/customized in Claude Code
- Context efficiency: skills only load fully when triggered, avoiding context bloat

# You'll NEVER Need Prompt Engineering Again with Meta-Prompting

Practical guide to meta-prompting: using an LLM to write prompts for other LLMs. Kashef shows how to describe what you want in plain language, ask Claude or ChatGPT to generate an optimized prompt, and then use that prompt as a reusable template. The core claim: manual prompt engineering is now largely solved; investing time in learning meta-prompting patterns returns more value than memorizing prompt syntax.

## Key Ideas
- Meta-prompting: give the LLM your goal in plain language, ask it to write the optimized prompt for you
- Describe the task, the output format, constraints, and the model you'll use — the LLM handles the rest
- Meta-prompts can include trigger words and conditional logic, making them closer to skills than one-off prompts
- Iterate on the generated prompt by feeding it back: "improve this prompt to handle edge case X"
- This approach scales: one good meta-prompt session produces a library of reusable prompt templates

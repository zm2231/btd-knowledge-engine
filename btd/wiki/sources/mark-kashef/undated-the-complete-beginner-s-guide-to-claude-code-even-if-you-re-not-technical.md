# The Complete Beginner's Guide to Claude Code (Even If You're Not Technical)

80-minute beginner course covering Claude Code from installation through first real projects. Designed explicitly for non-technical users. Covers terminal basics, CLAUDE.md setup, skills installation, plan mode vs. execution mode, and how to avoid common mistakes. Kashef's most accessible entry point for BTD users with no coding background who want to use Claude Code for personal or business workflows.

## Key Ideas
- Claude Code runs in a terminal but non-technical users can operate it entirely in plain English
- CLAUDE.md is the key file — write it once to give Claude persistent context about who you are and how you work
- Plan mode (shift-tab) lets you review Claude's intended actions before it executes — essential for beginners
- Start with a single contained task, not a big project; build confidence before adding complexity
- Skills let non-technical users invoke sophisticated workflows with a single slash command

# CLAUDE CODE FULL COURSE

72-minute course building a complete AI marketing officer from scratch using Claude Code. Ramonov covers installation, permissions, context basics, then builds two custom skills (content creation and brand voice), adds quality-gate hooks, and creates a multi-platform publishing pipeline using parallel sub-agents. The end workflow: one sentence triggers research, drafts a week of content, generates visuals, awaits approval, then schedules across all social platforms.

## Key Ideas
- Full setup walkthrough: installation, permissions, plan mode vs. execution mode, context management basics
- Custom skills are the building blocks — a content skill, brand voice skill, and quality-gate hooks layer together
- Hooks enforce quality before publishing: brand voice consistency checks, output format validation, approval gates
- Parallel sub-agents handle drafting, visual generation, and scheduling simultaneously for batch content production
- CLAUDE.md is the foundation — brand voice, samples, and workflow rules belong here before building skills

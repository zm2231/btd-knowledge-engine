# This Workflow Auto Posts in 7 Different Socials (Free Template)

18-minute walkthrough of an n8n workflow that researches viral news from Hacker News, writes a 30-second avatar video script, generates the video via HeyGen (with 11Labs voice and a faceless background from Blotato), and auto-publishes to 7 platforms: Instagram, YouTube, TikTok, Facebook, Threads, LinkedIn, Twitter. Free n8n template provided. The only setup required: HeyGen API key with avatar, 11Labs voice (optional), and Blotato API key for publishing.

## Key Ideas
- n8n orchestrates a research-to-publish pipeline: Hacker News → AI script → HeyGen avatar video → 7-platform publishing
- AI agent selects the most viral-potential article, writes a balanced 30-second monologue at sixth-grade reading level
- HeyGen's background removal + Blotato's faceless video URL creates avatar-over-video-background format
- Two caption variants generated: long-form (paragraph + 3 SEO questions + hashtags) and short-form (Twitter/Threads)
- Free template available in video description — only two nodes need configuration to run

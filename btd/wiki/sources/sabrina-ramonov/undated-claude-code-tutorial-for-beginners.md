# Claude Code Tutorial for Beginners

35-minute beginner tutorial walking non-technical users through installing Claude Code, setting up their first MCP server, and building a social media assistant. Ramonov explicitly addresses the anxiety of terminal-based tools and frames Claude Code as ChatGPT with the ability to get real work done. Ends with a practical first project: an AI social media assistant that can draft and schedule posts.

## Key Ideas
- Installation walkthrough: Claude.ai account, billing plan selection, Claude Code setup for absolute beginners
- MCP server setup demystified — connecting Claude Code to external tools like Notion, Airtable, Google Sheets
- First project: AI social media assistant as a contained, practical entry point
- Plan selection guidance: start Pro ($20/mo), upgrade to Max ($200/mo) when hitting usage limits
- Non-technical users can operate Claude Code entirely in plain English — no coding required

# OpenClaw AI Agent explained for beginners, including security issues

Short-form explainer on OpenClaw (Claude's agentic interface) aimed at complete beginners — what it is, why it's powerful, and critically, what the security risks are when giving an AI agent full computer access. Ramonov's characteristic non-technical framing: understand enough to use it safely, not enough to build it.

## Key Ideas
- OpenClaw gives Claude full computer access — file system, browser, terminal — not just a chat window
- The power is proportional to the risk: an agent with full computer access can cause real damage if misprompted
- Security basics: use CLAUDE.md to restrict what directories and tools the agent can touch
- Start with limited permissions, expand only when you understand what the agent is doing
- Non-technical users should understand the security model before running autonomous sessions

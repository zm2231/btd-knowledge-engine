# Claude Just Changed Content Creation Forever! (Tutorial)

Tutorial showing how to use Claude Code with the Remotion skill to create, edit, and publish videos entirely from the terminal — no paid video tools required. Covers motion graphics generation from a topic description, inserting web screenshots via Claude Chrome, editing existing talking-head videos (auto-captions, blooper removal), and publishing to social media via the Blotato MCP. Demonstrates a full end-to-end content production pipeline that can run from a single Claude Code session.

## Key Ideas
- Remotion skill installed via one command gives Claude Code free, local video creation and editing capability
- Workflow: describe video topic → Remotion generates motion graphics → Claude Chrome fetches web screenshots → insert assets
- Editing existing video: Claude transcribes via Whisper, identifies bloopers, adds captions, respects safe zones
- Brand system: separate files for brand voice, color kit, and assets ensure consistent style across all generated content
- Blotato MCP handles publishing — one command schedules the finished video to Instagram, TikTok, etc.

# AI vibe coding video games with Cursor AI, Claude Sonnet 3.7, and Three.js

Short-form demonstration of building a browser-based video game using Cursor AI with Claude Sonnet 3.7 and Three.js. Ramonov shows the vibe-coding loop: describe the game in plain English, let the AI generate the Three.js code, preview in browser, iterate. A curiosity/inspiration piece for builders exploring creative applications of vibe coding.

## Key Ideas
- Cursor + Claude Sonnet 3.7 can generate functional Three.js games from plain English descriptions
- Vibe coding for games is a low-stakes entry point for builders hesitant about technical tools
- Iteration loop: describe, preview, refine in natural language, repeat

# Google AI Edge Gallery

Google's official app for running Gemma 4 models on-device on iPhone, including multimodal and tool-calling demos. Willison notes this is the first official vendor app for on-device model testing and highlights the "skills" demo showing HTML-page-backed tool calls — directly relevant to anyone exploring local LLM deployment or agent tool-use patterns.

## Key Ideas
- Gemma-4-E2B (2.54GB) runs fast and is "genuinely useful" on an iPhone — local LLM quality has crossed a practical threshold for mobile
- The "skills" demo exposes tool calling where each skill is a standalone HTML page — a sandboxed artifact pattern similar to Claude Artifacts
- First official vendor app for on-device model testing normalizes local inference for non-technical users
- Missing: persistent conversation logs — conversations are ephemeral, a UX gap for serious use

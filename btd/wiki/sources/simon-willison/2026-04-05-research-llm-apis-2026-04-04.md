# research-llm-apis 2026-04-04

Willison documents using Claude Code to research raw API behavior across Anthropic, OpenAI, Gemini, and Mistral — reading their Python SDKs and generating curl commands to capture streaming and non-streaming JSON responses. The goal is designing a new abstraction layer for his LLM library that can handle vendor-specific features like server-side tool execution.

## Key Ideas
- Using Claude Code to read SDK source code and generate API research scripts is a practical agentic workflow for library design
- Different LLM vendors now have features (server-side tool execution, streaming variants) that resist a single unified abstraction
- README-driven development and agent-assisted research are complementary techniques for solo library maintainers
- Capturing raw API JSON responses across vendors reveals divergences not documented in higher-level SDKs

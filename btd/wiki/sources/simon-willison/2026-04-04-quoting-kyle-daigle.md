# Quoting Kyle Daigle

GitHub's COO shares platform activity numbers that put AI-driven development growth in concrete terms: commits are on pace for 14 billion in 2026 (up from 1 billion in all of 2025), and GitHub Actions minutes have more than doubled since 2025. A data point on how much AI-assisted coding has accelerated the volume of software output.

## Key Ideas
- GitHub commit volume: 275 million/week in 2026, on pace for 14B/year vs. 1B in all of 2025
- Actions minutes doubled from 1B/week (2025) to 2.1B/week — indicating more automated pipelines running more code
- Raw volume of software output has increased faster than at any point in GitHub's history

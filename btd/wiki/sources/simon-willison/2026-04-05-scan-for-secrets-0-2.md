# scan-for-secrets 0.2

Feature release adding streaming output, multi-directory scanning, single-file scanning, verbose mode, and a Python API. Demonstrates how agentic development enables rapid iteration — multiple usability improvements shipped the same day as the initial release.

## Key Ideas
- Streaming results as found (rather than at end) is critical UX for scanning large directories — a design improvement obvious only after using the tool
- Exposing both a CLI and a Python API in the same release gives the tool maximum reuse value
- Same-day iteration from 0.1 to 0.2 is only practical with AI-assisted implementation

# Cleanup Claude Code Paste

A micro-tool Willison built to strip the `❯` prompt character and fix wrapped-line whitespace when copying prompts out of the Claude Code terminal. Useful for anyone publishing Claude Code session transcripts. A good example of building tiny single-purpose tools to smooth workflow friction.

## Key Ideas
- Copying prompts from Claude Code terminal introduces whitespace artifacts that need cleaning before publishing
- Tiny utility tools like this are worth building — the effort is low, the friction reduction is real
- Illustrates the "scratch your own itch" pattern that becomes practical when AI can ship a one-page tool in minutes

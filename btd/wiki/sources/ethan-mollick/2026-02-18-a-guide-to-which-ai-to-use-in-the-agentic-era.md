# A Guide to Which AI to Use in the Agentic Era

Mollick's most complete guide to the new three-layer landscape of AI: Models (the underlying intelligence), Apps (the products you interact with), and Harnesses (the systems that let models take autonomous action). The same model behaves radically differently depending on the harness — Claude in a chat window vs. Claude Code running autonomously for hours are fundamentally different tools. He covers the full current landscape: chatbot interfaces, coding agents, Claude Cowork (Code for non-technical work), NotebookLM, and the security-nightmare-but-telling OpenClaw.

## Key Ideas
- Model, App, Harness is the new framework: the harness increasingly matters more than model differences for most real work
- Free models are optimized for chat speed and personality, not accuracy — always select advanced thinking models for real work
- Claude Cowork represents "Claude Code for knowledge workers": local file access, app integrations, mobile dispatch interface
- NotebookLM solves a different problem — making sense of a pile of documents — with research, Q&A, slides, and podcast output
- An AI that does things is fundamentally more useful than an AI that says things
